## üîπ **Spread Operator (`...`)**

**Use it to _expand_ values** ‚Äî most often used in arrays, objects, or function calls.

### ‚úÖ **Examples:**

### 1. **In Arrays:**
```js
const a = [1, 2];
const b = [...a, 3, 4]; 
console.log(b); // [1, 2, 3, 4]
```

### 2. **In Objects:**
```js
const user = { name: "Daniel" };
const fullUser = { ...user, age: 30 };
console.log(fullUser); // { name: "Daniel", age: 30 }
```

### 3. **In Function Calls:**
```js
const nums = [1, 2, 3];
Math.max(...nums); // 3
```

---

## üî∏ **Rest Operator (`...`)**

**Use it to _collect_ values into an array or object** ‚Äî often used in function parameters or object destructuring.

### ‚úÖ **Examples:**

### 1. **In Function Parameters:**
```js
function sum(...args) {
  return args.reduce((a, b) => a + b, 0);
}
sum(1, 2, 3); // 6
```

### 2. **In Array Destructuring:**
```js
const [first, ...rest] = [10, 20, 30];
console.log(first); // 10
console.log(rest);  // [20, 30]
```

### 3. **In Object Destructuring:**
```js
const user = { name: "Daniel", age: 30, city: "HCM" };
const { name, ...details } = user;
console.log(name);    // "Daniel"
console.log(details); // { age: 30, city: "HCM" }
```

---

### ‚ö†Ô∏è Quick Tip:
Same syntax (`...`), **but usage depends on context**:
- **Spread**: _on the right_ side ‚Üí expands things
- **Rest**: _on the left_ side ‚Üí gathers things
