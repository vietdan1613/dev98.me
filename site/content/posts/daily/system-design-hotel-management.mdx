## 4ï¸âƒ£ Routing & Rendering Strategy (KEY SECTION)

### App Router structure

```
/app
  /search
    page.tsx        // SSR
  /hotel/[id]
    page.tsx        // SSG + ISR
  /booking
    page.tsx        // CSR
  /confirmation
    page.tsx
```

### Rendering choice

| Page         | Strategy  | Why                    |
| ------------ | --------- | ---------------------- |
| Search       | SSR       | SEO + fresh data       |
| Hotel detail | SSG + ISR | Fast + scalable        |
| Booking      | CSR       | Secure & user-specific |
| Payment      | CSR       | Sensitive data         |

ğŸ’¬ Interview gold:
